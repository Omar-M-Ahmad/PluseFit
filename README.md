# ✨ Overview

PulseFit is a production-ready modern Gym & Fitness landing page built using the latest web technologies.

The project focuses on:

- ⚡ Performance-first architecture
- 🌍 Full Arabic & English support
- 🌙 Dark / Light mode
- 📱 Fully responsive layout
- 🎯 Pixel-perfect modern UI
- 🧩 Clean scalable architecture
- ✨ Smooth animations & transitions
- 🛡️ Production-ready structure

# ⚙️ Getting Started

## 1️⃣ Clone Project

```bash
git clone https://github.com/Omar-M-Ahmad/PluseFit.git
```

---

## 2️⃣ Install Dependencies

```bash
pnpm install
```

---

## 3️⃣ Run Development Server

```bash
pnpm dev
```

---

## 4️⃣ Build Production

```bash
pnpm build
```
